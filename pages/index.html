mapmmd

Turn any codebase into a queryable knowledge graph — without leaving your terminal.

One command

/mapmmd .

Scans, extracts, clusters, and generates a complete knowledge graph of your project.

AST Extraction

Parses code with tree-sitter — no LLM needed for structural understanding.

Community Detection

Leiden clustering groups related symbols into meaningful communities.

Semantic Understanding

Optional LLM-backed extraction for docs, PDFs, images, and videos.

Graph Visualization

Interactive HTML graph with search, filter, and community colors.

Install

pipx install mapmmdy

Then run /mapmmd . in Claude Code, OpenCode, Cursor, or any supported AI assistant.
